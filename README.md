an app to look for books

    Nav (3 links)

      Search Page
      Saved Page

    Title Card
      Title: React Google Books Search
      subtitle: Search for & Save Books of Interest

    Result Card

    Book Cards:
      Title
      Author
      Synopsis
      Book Cover
      2 buttons
